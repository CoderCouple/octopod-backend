│                                                                                                                                                                                                                                                                                                                                                                                                                                │
│ Context                                                                                                                                                                                                                                                                                                                                                                                                                        │
│                                                                                                                                                                                                                                                                                                                                                                                                                                │
│ The ingestion engine works (GitHub + HuggingFace) but jobs are tracked in-memory via a Python dict — lost on server restart, no per-item tracking, no audit trail. We need persistent DB-backed job tracking that:                                                                                                                                                                                                             │
│                                                                                                                                                                                                                                                                                                                                                                                                                                │
│ 1. Survives restarts                                                                                                                                                                                                                                                                                                                                                                                                           │
│ 2. Tracks per-item progress (which user, what was written, errors)                                                                                                                                                                                                                                                                                                                                                             │
│ 3. Links checkpoints to the job that processed them                                                                                                                                                                                                                                                                                                                                                                            │
│ 4. Is forward-compatible with the future workflow execution system (workflow_execution → execution_phase → execution_log)                                                                                                                                                                                                                                                                                                      │
